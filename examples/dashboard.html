GUIM Spec Reference
Tag Required attrs Optional attrs
[card] — title, width="full"
[grid] — columns (default 2)
[metric /] label, value trend ("+X%" · "up" · "down")
[chart /] type, data (0–100) label
[table /] headers, rows — (rows separated by ";")
[tasktracker /] tasks — (tasks separated by ",")
[weather /] city, temp condition
[list] [listitem /] listitem: text —
[button /] label variant ("primary" · "secondary")
